# 영화 예매 시스템
### 요구사항
사용자는 영화 예매 시스템을 이용해 쉽고 빠르게 영화를 예매할 수 있다

### 용어
사용자가 실제로 예매하는 대상은 영화가 아니라 상영
* 영화
    * 제목, 상영시간, 가격정보 등
* 상영
    * 실제로 관객이 영화를 관람하는 사건. 상영일자, 시간, 시간 등
* 할인액을 결정하는 두가지 규칙
    * 할인 조건 - 가격의 할인 여부를 결정
        * 순서조건 - 매일 몇번째 상영 영화
        * 기간조건 - 요일, 시작~종료시간
    * 할인 정책
        * 비율 - 일정 비율 (%)
        * 금액 - 일정 금액
